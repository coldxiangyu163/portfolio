## 📦 本地运行

```bash
# 克隆仓库
git clone https://github.com/coldxiangyu163/portfolio.git

# 进入目录
cd portfolio

# 使用任意 HTTP 服务器运行
python3 -m http.server 8000
# 或
npx serve
```

访问 http://localhost:8000
